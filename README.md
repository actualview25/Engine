# Actual View Engine 🏗️

منصة متكاملة للجولات الافتراضية، الملاحة التفاعلية، وإدارة مشاريع البناء.

## ✨ الميزات الرئيسية

- 🎯 **نظام نقاط متقدم مع انتقالات سلسة
- 📏 **أدوات قياس كاملة** (مسافة، مساحة، حجم، زوايا)
- 🏗️ **دعم BIM و IFC** - استيراد ملفات البناء
- 📐 **دعم CAD** - DXF, SVG, GeoJSON
- 📊 **نظام كميات (BOQ)** - حسابات وتكاليف وتصدير
- 🎙️ **أوامر صوتية** - تحكم كامل بالصوت
- 🌐 **تكامل APIs** - مزامنة مع BIM360 و WebSockets
- 📄 **تقارير متقدمة** - PDF, Excel, HTML, Snapshots
- 🚀 **أداء فائق** - Raycaster محسن، LOD، Web Workers
